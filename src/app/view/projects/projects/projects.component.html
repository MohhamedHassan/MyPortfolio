Project Name:

GoodReads

About:

It is a book-sharing site. Each user can create an account and add information about his book, such as the book's picture, the name of the book, and the description of the book. He must have an account to do so. All users' books and reviews appear on the home page. Each user can rate the books of others from one to five. And the user can see his books and personal information on his personal page

tools:

html5, css3, bootstrap, scss, angular +2, angular material, ng bootstrap, ngx toaster ,
